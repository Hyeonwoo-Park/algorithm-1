2. 모든 순열이 정해졌으면 이닝의 수(N)만큼 반복문을 반복합니다.
3. p1, p2, p3은 각각 1 , 2, 3루로 switch case로 타자들을 이용해가며 score 점수를 확인합니다.
4. out이 3번 됐으면 반복문을 탈출하고 다음 이닝으로 넘어갑니다. (이때 순서는 유지)
