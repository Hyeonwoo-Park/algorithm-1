## 👩‍🏫 문제 풀이
- 10 * 10 배열을 모두 돌며,
    - 1을 만나면
        - 해당 칸을 시작으로 1 ~ 5 까지의 정사각형이 들어가지는지 확인.
        - 들어기자면 채우고 다음칸 확인.
        - 돌아오면 해당칸 다시 비움.
    - 모든 색종이가 없이지면 (sum == 0) 현재 depth로 minimum 갱신.

```java
loop: 
for (int i = 0; i < 10; ++i) {
    for (int j = 0; j < 10; ++j) {
        if (board[i][j] == 1) {
            for (int k = 1; k <= 5; ++k) {
                if (papers[k] > 0 && fillable(board, i, j, k)) {
                    ...
                }
            }
            break loop;
        }
    }
}
```
- 어떤 칸 ij에서 1 ~ 5까지의 가능성을 모두 확인했다면, 이후에 다시 돌아올 일이 없어야 함.
