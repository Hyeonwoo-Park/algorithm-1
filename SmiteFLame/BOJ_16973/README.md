# BOJ 16973번 직사각형 탈출

## 🌈 풀이 후기
BFS를 활용하여 문제를 풀었습니다.<br>
이동할 수 있는 조건을 좀 더 추가하여서 간단하게 풀었던 문제입니다<br>



## 👩‍🏫 문제 풀이
<br>

### 변수
- N, M(int): 격자판 크기
- H, W(int): 직사각형 크기
- Sr, Sc(int): 시작 위치
- Fr, Fc(int): 도착 위치
- map (int[][]) : 전체 위치

### 원리
1. 시작 위치부터 BFS를 시작합니다.
2. 상하좌우를 확인하면서 전체 범위가 이동할 수 있는지 직사각형 전부를 조사합니다.
3. 도착 위치에 가면 BFS를 종료하고 결과값을 출력합니다.
