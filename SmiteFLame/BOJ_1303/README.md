# BOJ 1303번 전쟁-전투

## 🌈 풀이 후기
BFS를 활용하여 문제를 풀이 했습니다.
처음부터 조사가 되지 않은 지역은 검사를 진행하고
검사가 완료된 지역은 'X'로 로 변경했습니다.

## 👩‍🏫 문제 풀이
<br>

### 변수
- N(int): 가로크기
- M(int): 세로크기
- NEWS(int[][]): 상하좌우 값
- map(char[][]): 데이터의 저장

### 원리
1. 0, 0 부터 M, N 까지 전수 조사
2. 만약 'X'값이 아니라면 BFS 실행, BFS을 처리한 값은 'X'처리
3. 각각의 결과 값을 각각의 sum값에 추가
